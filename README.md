本プロジェクトは、未経験からWebアプリケーション開発を学ぶために個人で開発している学習用プロジェクトです。
単に動くものを作るだけでなく、設計・セキュリティ・保守性を意識した開発を目標としています。
将来的には実際にテスト運用を行い、日本の英語教育に貢献できるサービスへ発展させることを目標としています。

# 目的
本アプリは、小学生・中学生を対象に、英単語のスペル学習を支援することを目的としています。
- 生徒は英単語のスペルを学習し、テスト形式で実力を確認できます
- 学習を競争形式にすることで、読解力や記述力の向上を目指します
また、教師向けに以下の管理機能を想定しています。
- 授業グループの作成と管理
- 年度ごとの生徒リセットおよび新規招待
- 教科書の章ごとに対応した練習やテストの配信
- クラス全体のスコア一覧の確認
生徒は自分のスコアのみ確認可能とし、他者のスコアは閲覧できません。
一方で、教師は授業内の全生徒の成績を把握できます。

将来的には、クラス単位に留まらず、校内や地域単位でのランキング機能を実装し、
生徒同士が健全に競争しながら成長できる環境を適用したいと考えています。

※本アプリは、スコアによる他者への誹謗中傷やいじめ行為を明確に禁止し、利用目的に反する行為として扱います。

# 使用技術
## Python
-本プロジェクトのメイン言語
-バックエンドおよびデータベース処理を担当
-Flaskフレームワークを使用

## SQL
-開発環境ではSQLite3を使用
-本番環境ではPostgreSQLへの移行を想定
-ユーザー情報、スコア、英単語データなどを管理

## JavaScript
-Dashboard、Test、Register画面などで使用
-非同期処理によるUX向上を目的

## HTML/CSS
-シンプルで分かり易いUIを重視
-デザイン面は今後、知見のある方と相談しながら改善予定
